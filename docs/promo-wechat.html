✦ 设计要点
关键决策

消息聚合

多图 2 秒防抖收集，5 分钟内文字触发合并查询。超时附件自动丢弃。

中途截流

处理中发新消息 → 终止当前查询 → 合并上下文重新发送。

智能分割

段落边界优先切分，保持 markdown 完整。三层降级：换行→句号→硬切。

日志脱敏

Bearer token、bot token、context token、API key 全部正则脱敏后写入。

限频保护

每用户 2.5s 最小间隔。ret:-2 时退避 5s→10s→30s。typing ticket 缓存 24h。

崩溃恢复

启动时检测残留 processing 状态自动重置。launchd KeepAlive 保证自动重启。
