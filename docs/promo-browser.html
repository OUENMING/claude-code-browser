✦ 四大 Content Script
Extension 核心
accessibility-tree.js
WeakRef 元素映射，30+ ARIA 角色检测。interactive 模式（省 token）和 all 模式。状态显示（禁用/选中/只读），字符上限截断可续接。
page-bridge.js
正文自动检测。表单填充走 prototype setter 兼容 React/Vue。事件序列：InputEvent('input', inputType:'insertText') + Event('change')。多词元素搜索。
auto-capture.js
HTML→Markdown 转换。过滤 <50px 装饰图标。表格限制 10 行 8 列，嵌套列表递归，details/summary 支持。
visual-indicator.js
Shadow DOM 叠加层 UI。元素绿色脉冲高亮，loading/completed/error 状态徽章，Agent 操作中脉冲边框 + 停止按钮。7 种 message 类型。
