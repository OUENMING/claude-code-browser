工具清单

🧭 导航

navigate

导航到指定 URL，支持 "back"/"forward" 前进后退。
必填: url

📖 页面读取

read_page

获取可访问性元素树，每个元素带 [ref_N] 标识符。filter="interactive" 仅交互元素（省 token），"all" 全部。
可选: filter max_chars ref_id tabId

find

按关键词搜索元素，匹配 text/aria-label/title/role。返回 ref 列表供 computer/form_input 使用。
必填: query · 可选: max_results

🖱 交互

computer

鼠标/键盘/截图交互的核心工具。13 种 action：left_click, right_click, double_click, triple_click, type, screenshot, wait, scroll, scroll_to, key, left_click_drag, hover, zoom
必填: action · 选填: ref coordinate text quality modifiers

📝 表单

form_input

用 ref 设置表单字段值。React/Vue 受控组件兼容（prototype setter）。select 用 option text，checkbox 用 boolean。
必填: ref value

📄 内容提取

get_page_text

纯文本提取（textContent）。最完整，不丢内容。自动选 article/main 等语义容器。适合社交媒体、复杂 SPA。
可选: max_chars (默认 50000)

get_page_markdown

结构化 Markdown 提取。保留标题(#)、链接、代码块、表格、图片。过滤 <50px 装饰图标。适合博客、文档、产品页。
可选: max_chars (默认 50000)

🔧 JS & 调试 & 标签页

javascript_tool

在当前页面执行 JavaScript。三种尝试策略：scripting → CDP 同步 → CDP async IIFE。
必填: text

read_console_messages

读取浏览器控制台消息，支持 onlyErrors/pattern 过滤、limit 截断、clear 清空。
必填: tabId · 可选: onlyErrors pattern clear limit

read_network_requests

读取 HTTP 网络请求，支持 urlPattern 正则过滤。
必填: tabId · 可选: urlPattern clear limit

tabs_context

列出所有打开的标签页（含 active 标记、URL、标题）。

tabs_create

创建新的空白标签页。

技术细节

CDP 输入

通过 chrome.debugger 发送 CDP 命令，走真实浏览器输入管线
// 点击流程 Input.dispatchMouseEvent mouseMoved → 50ms → Input.dispatchMouseEvent mousePressed → 50ms → Input.dispatchMouseEvent mouseReleased // 中文输入 Input.insertText { text: ch } // 逐字符
✱ 每个 ASCII 字符 3 次 CDP 调用 (keyDown→char→keyUp) + 20ms 延迟
✱ 中文逐字符 Input.insertText + 10-30ms 随机延迟

元素定位

WeakRef 元素映射 + ARIA 角色推导，不依赖 CSS selector
// ref 系统 elementMap: Map<"ref_1", WeakRef> reverseMap: WeakMap<Element, "ref_1"> // 角色推导 getRole(el): aria-role → tagName → onclick? getAccessibleName: aria-label → aria-labelledby → title → placeholder → textContent // 元素树输出 [ref_48] textbox "搜索小红书" [ref_13] combobox "马龙出征全锦赛"

保活策略

双轨保活：
① setInterval + chrome.storage.local.set 每 25s
② chrome.alarms 每 0.5min 后备唤醒

MV3 Service Worker 30s 空闲被回收 → alarms 唤醒 → 自动重连
